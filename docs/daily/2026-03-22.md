## HarborOS 联调指南

### 快速理解

HarborBeacon 是「IM → NAS 操作 → 回复」的中间层。HarborOS 只需关心两个接口：

```
用户 IM ──→ [HarborBeacon 管道] ──→ MCP Tool Call ──→ [HarborOS 执行器] ──→ 结果 ──→ 回复用户
```

```bash
cd /path/to/HarborNAS-LocalAgent-Project

```

### 第二步：消息入口（选择一种）

**方式 A — Long Connection（无需公网 IP，推荐）**

```python
from harborbeacon.long_connection import Gateway, FeishuWsConfig
from harborbeacon.dispatcher import Dispatcher

gateway = Gateway(on_message=dispatcher.handle)
gateway.register_feishu(FeishuWsConfig(app_id="xxx", app_secret="xxx"))
gateway.start_all()
```

**方式 B — Webhook（需要公网 IP 或内网穿透）**

```python
from harborbeacon.webhook import WebhookReceiver

receiver = WebhookReceiver(dispatcher=dispatcher, host="0.0.0.0", port=9000)
receiver.start()
```

### 第三步：HarborOS 执行器对接

HarborBeacon 调用 HarborOS 的路径：`Dispatcher → McpServerAdapter → Runtime → Router → Executor`

如果你要新增一个 HarborOS 操作，只需实现 Executor 接口：

```python
from orchestrator.contracts import Route, ExecutionResult, StepStatus

class MyHarborOSExecutor:
    @property
    def route(self) -> Route:
        return Route.MIDDLEWARE_API  # 或 HARBOR_CLI

    def is_available(self) -> bool:
        return True  # 检查 HarborOS 是否可达

    def execute(self, action, *, task_id, step_id) -> ExecutionResult:
        # 调用 HarborOS API
        result = call_harbor_os(action.resource)
        return ExecutionResult(
            task_id=task_id,
            step_id=step_id,
            executor_used="harbor_os_api",
            status=StepStatus.SUCCESS,
            result_payload=result,
        )
```

然后注册到 Router：

```python
from orchestrator.router import Router

router = Router([MyHarborOSExecutor()])
```

### 第四步：如果需要解析群消息 / 富媒体

今天新增的字段，`InboundMessage` 现在携带更多上下文：

```python
msg.message_id   # "om_xxx" — 平台消息ID，用于去重和回复
msg.chat_type    # ChatType.P2P 或 ChatType.GROUP
msg.chat_id      # "oc_xxx" — 会话ID，群聊时存在
msg.mentions     # ["@bot_key"] — 消息中的 @提及
msg.attachments  # [Attachment(type=IMAGE, content="img_key123")]
```

Dispatcher 已自动处理：
- **去重**：相同 message_id 不会重复执行
- **群过滤**：群里闲聊不触发，只有 @机器人 / 问句 / 请求动词 才响应
- **会话隔离**：群聊用 chat_id 做 session key，不同用户在同一群共享上下文

### 2. 品牌重命名：HarborClaw → HarborBeacon
