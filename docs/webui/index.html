扫码后的真实接入流程

二维码只负责打开后台，真正的消息桥接接入发生在手机端表单
  1. 用户用手机扫描设备上的二维码，手机浏览器打开这台 Hub 的移动配置页。
  2. 用户在手机上填入自己创建的消息桥接应用的 app_id 和 app_secret。
  3. 后台即时验证兼容 Open API 与应用信息，保存成功后 Hub 获得统一 IM 消息入口。
  4. 之后用户回到 IM 里，就可以直接发“扫描摄像头”“看看客厅摄像头”等命令。
  5. 后台页面继续负责策略、设备库和审计，不抢用户主入口。
